Formulários em Steps

Formulários divididos em etapas com collapseGroup

Para formulários divididos em passos, utilizamos o componente collapse do LocawebStyle, agrupado em um collapseGroup

As diferenças principais são o modo como se navega no collapse, e as opções são:

Steps bloqueados

Só é possível exibir um step de cada vez, com navegação por botões dentro de cada step

01. Passo 1

Ir para o passo 2

02. Passo 2

Voltar para o passo 1 Ir para o passo 3

03. Passo 3

Voltar para o passo 2 Finalizar

Revise seus dados e clique em enviar

Voltar para o passo 3

Steps livres

É possível exibir mais de um step de cada vez, com navegação por botões dentro de cada step ou pelos títulos de cada step

01. Passo 1

Ir para o passo 2

02. Passo 2

Voltar para o passo 1Ir para o passo 3

03. Passo 3

Voltar para o passo 2Finalizar
